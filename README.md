# Quinigem

Gem to scrap the Quiniela results from http://www.loteria-nacional.gov.ar

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'quinigem'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install quinigem
